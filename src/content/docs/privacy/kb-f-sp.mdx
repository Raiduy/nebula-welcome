Using the [Knowledge Bases](/welcome/knowledgeBases/) and [Folders](/welcome/folders/) functionalities in Nebula requires you to upload files to the server. These files are saved and  backed up. As such, DO NOT upload sensitive information here.

[System Prompts](/welcome/advancedParams#system-prompt/) that are set [globally](/welcome/modifyingAdvancedParams#global-modification) and [per model](/welcome/modifyingAdvancedParams#per-model-modification) are saved to the server. DO NOT upload sensitive information here.
This information is saved to the server because it is meant to be reused. We chose to keep this information to avoid forcing you to recreate it every time you log in.
