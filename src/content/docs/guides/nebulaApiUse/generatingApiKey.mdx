<Aside>Nebula is **OpenAI compatible**. You can use the OpenAI package from your favorite programming language to interact with Nebula.</Aside>

The Nebula platform also allows for API access to the models, knowledge bases, and file uploads.
See [here](https://docs.openwebui.com/reference/api-endpoints) the original documentation

## Generating an API key
<Steps>
1. Click on your **profile picture**, either on the bottom left or top right corner of your screen.

2. Click on **Settings**.

3. Click on **Account**.

4. Click on **Show**, to the right side of the **API Keys**.

5. Create your keys.
</Steps>
