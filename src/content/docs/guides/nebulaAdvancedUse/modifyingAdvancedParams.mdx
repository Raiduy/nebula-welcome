The advanced parameters can be accessed and modified in three locations depending on where you want the parameters to be applied.

## Global modification

Modifying the parameters here will cause the parameters to be applied to all the prompts you send, unless you use a model with modified parameters, or modify the parameters for a specific prompt.
   
   ● Go to **Settings -> General**
   
   ● To modify the System Prompt, write your text in the system prompt box
   
   ● To modify the other parameters, **Advanced Parameters -> Show**

## Per model modification

Modifying the parameters here will cause the parameters to be applied to all the prompts you send using the specific model, unless you modify the parameters for a specific prompt.
  
  ● [Create your own model]() or [edit your model]()
  
  ● To modify the System Prompt, write your text in the system prompt box
  
  ● To modify the other parameters, **Advanced Parameters -> Show**

## Per prompt modification

Modifying the parameters here will cause the parameters to be applied to this conversation only.
  
  ● Before sending the prompt to Nebula, click on the <img src="/src/assets/settingsBtn.png" height="11" style="margin:0; display: inline;"/> button on the top right of your screen
  
  ● A window will pop up on the right side of the screen with all of the parameters
